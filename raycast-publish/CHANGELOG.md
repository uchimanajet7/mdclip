# MdClip Changelog

## [Initial MdClip Release] - {PR_MERGE_DATE}

- Add Markdown Source 1, Markdown Source 2, Markdown Source 3, and All Markdown Sources commands.
- Add Markdown Source enable switches to exclude unused sources without removing folder settings.
- Add recursive Markdown file listing from enabled and configured Markdown Source folders.
- Add actions to copy raw content, copy expanded content, open files, show files in Finder, and toggle preview.
- Add copy-time placeholder expansion for reusable Markdown contents, including current date, time, time zone, UUID, and clipboard values.
- Add preview pane and sorting by updated time, file name, and relative path.
- Keep local Markdown files as the source of truth; MdClip does not create, edit, move, rename, or delete those files.
