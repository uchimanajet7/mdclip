# Local Copy Blocks Changelog

## [Initial Release] - {PR_MERGE_DATE}

- Add Block Set 1, Block Set 2, Block Set 3, and All Block Sets commands.
- Add Block Set enable switches to exclude unused Block Sets without removing folder settings.
- Make Block Set settings easier to scan in Preferences.
- Add recursive Markdown-backed text block listing from enabled and configured Block Set folders.
- Add actions to copy raw content, copy expanded content, open files, show files in Finder, and toggle preview.
- Add copy-time Dynamic Placeholders for reusable blocks, including current date, time, time zone, UUID, and clipboard values.
- Add preview pane and sorting by updated time, file name, and relative path.
